Hướng dẫn

Bật Associated Domains trên iOS với applinks:spcable.github.io và cấu hình Android App Links theo assetlinks.json. Lưu ý: iOS/Android yêu cầu file xác thực ở gốc domain (https://spcable.github.io/.well-known/...). Nếu site chạy ở đường dẫn /tcsc.github.io/, cần host AASA/assetlinks ở gốc domain hoặc dùng domain riêng.
